# shopify-challenge

Step 1: Download the Repository
-Option 1: Download ZIP (https://github.com/bienskie/shopify-challenge)
-Open the GitHub repository.
-Click the Code button.
-Select Download ZIP.
-Extract (unzip) the downloaded file on your computer.

Step 2: Upload the Theme to Shopify
-Log in to your Shopify Admin.
-Go to Online Store → Themes.
-Click Add theme.
-Choose Upload zip file.
-Select the ZIP file of the theme.
-Wait for the upload to finish.

Step 3: Preview the Theme
-In Online Store → Themes, find the uploaded theme.
-Click Actions → Preview.
-Check that the pages load correctly.

Step 4: Publish the Theme
-Click Actions → Publish.
-Confirm the action.
